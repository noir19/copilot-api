# 2026-04-09 Dashboard Layout Reflow Review

## 范围

- 调整 dashboard 头部信息层级
- 调整 overview 页面的主次卡片布局比例

## 改动

- 将 `刷新按钮` 和 `已记录请求数` 提到头部上层，形成独立操作条
- 将说明文案下沉到标题区，避免侵占操作区域
- 保留现有视觉语言，不改组件体系
- 将 overview 从模糊比例网格改成明确的 12 栏布局
  - `模型分布` 7 栏
  - `Copilot 配额` 5 栏
  - `模型明细` 5 栏
  - `最近请求` 7 栏

## 结果

- 头部的操作、状态、说明分层更清楚
- `模型分布` 与 `Copilot 配额` 的比例不再像误差导致的视觉倾斜
- 下半区的主次关系也更稳定

- `bun run lint`
- `bun run typecheck`
- `bun run build`
